# Release v5

## 版本说明

本次公共版本命名为 `v5`，但对齐的是本地最终的 **V10** 路线。由于中间的 v5-v9 没有逐版公开上传，因此 GitHub 版本号直接从历史公开版本跳转到当前的 `v5`。

## 更新内容

- 完成仓库瘦身，移除训练数据、历史中间产物、重复模型和本地缓存目录
- 合并说明类文档，重写 README，使其更符合 GitHub 标准项目展示方式
- 保留训练、评估、推理、前后端和论文材料的完整入口

- 补充 `CHANGELOG`、`PROJECT_GUIDE`、`Thesis Materials` 和 GitHub 协作模板
